## Tenant vs Site Collection App Catalog

| | Tenant App Catalog | Site Collection App Catalog |
|---|---|---|
| Scope | All sites in the tenant | One site collection |
| Who manages it | SharePoint Admin | Site Collection Admin |
| URL | `/sites/appcatalog` | `/sites/yoursite/_catalogs/AppCatalog` |
| Use case | Organisation-wide solutions | Team or project-specific solutions |

## What uploading a .sppkg does

When you upload a `.sppkg` file to the App Catalog:

1. SharePoint extracts the manifest to learn what the solution contains
2. It shows you a trust dialog — you grant or deny tenant-wide deployment
3. If approved with tenant-wide deployment, the web part becomes available on every site immediately
4. If not tenant-wide, site admins must still add it to their site from the Site Contents page
