# What is React?

> Full lesson content coming in Task 12.

React is a JavaScript library for building user interfaces. It was created by Facebook (now Meta) and open-sourced in 2013.

## Why React?

React solves the problem of keeping your UI in sync with your data. Instead of manually updating the DOM every time something changes, React does it for you.
